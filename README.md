# Phone Checklist Program

Este repositorio incluye un programa sencillo en Python para realizar un checklist de pruebas en un celular.

## Requisitos
- Python 3.x

## Uso
Ejecute el archivo `phone_checklist.py` desde una terminal:

```bash
python3 phone_checklist.py
```

El programa le preguntará por distintas funciones del dispositivo (pantalla, botones, cámara, etc.) y usted deberá responder `s` para sí o `n` para no. Al finalizar mostrará un resumen y ofrecerá la opción de guardar un reporte en `reporte_pruebas.txt`.
